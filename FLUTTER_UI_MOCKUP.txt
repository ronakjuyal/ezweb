MULTI-IMAGE PICKER INTERACTION:

1. Click "Add More Images" button
2. Multi-select image picker opens (native gallery)
3. User selects 2 images
4. Loading indicator appears
5. Images upload to S3 one by one
6. Grid updates showing new images:

   ┌──────────┐  ┌──────────┐  ┌──────────┐
   │   📷     │  │   📷     │  │   📷     │
   │  Image1  │  │  Image2  │  │  Image3  │
   │    ❌    │  │    ❌    │  │    ❌    │
   └──────────┘  └──────────┘  └──────────┘

   ┌──────────┐  ┌──────────┐
   │   📷     │  │   📷     │     5/5
   │  Image4  │  │  Image5  │
   │    ❌    │  │    ❌    │
   └──────────┘  └──────────┘

7. Button becomes disabled (max reached)
8. Success snackbar: "2 image(s) uploaded successfully"

To remove an image:
- Click ❌ on any image
- Image removed from grid
- Counter updates: "4/5"
- Upload button re-enabled

✅ Multi-Images:
   - 3-column responsive grid
   - Individual remove buttons
   - Counter display (X/maxImages)
   - Batch upload capability
   - Disabled when limit reached
   - Loading feedback
   - Success messages

✅ Advanced Section:
   - Collapsed by default
   - Settings icon indicator
   - Smooth expand/collapse animation
   - Contains any schema field types
   - Keeps main form clean
   - Power users can still access

✅ Overall UX:
   - Clean, organized layout

   - Clear visual hierarchy
   - Responsive design
   - Helpful descriptions
